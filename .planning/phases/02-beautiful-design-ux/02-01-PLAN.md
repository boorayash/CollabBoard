# Plan: 02-01 - Foundation & CSS Tokens

**Goal**: Establish the base CSS design system, typography hierarchy, and global animated background without breaking existing functionality.

## Context
See the Phase 2 Implementation Plan for exact color codes and glassmorphism values.

## Tasks

### 1. Typography & Global CSS
<action>
- Modify `client/src/index.css`:
  - Import Google Fonts: Inter and Outfit.
  - Define CSS variables: `--font-heading: 'Outfit', sans-serif;` and `--font-body: 'Inter', sans-serif;`.
  - Define dual accent colors: `--color-primary: #7C3AED;` and `--color-secondary: #06B6D4;`.
  - Define glassmorphism tokens EXACTLY as:
    ```css
    --glass-bg: rgba(255, 255, 255, 0.05);
    --glass-border: rgba(255, 255, 255, 0.1);
    --glass-blur: 12px;
    ```
  - Hide horizontal and vertical scrollbars site-wide while keeping scroll functionality (`::-webkit-scrollbar { display: none; }`).
</action>
